Step by step

  1. Upload the label — one file, or the front and back as two separate files. Drag the file into the First file box, or click the box to pick a file from your computer. If the back of the label is its own file, put it in the Second file box.
  2. The tool reads the label and fills in the Label application fields for you. It also works out the product type (wine, malt beverage, or distilled spirits) and whether the product is imported. If it gets either wrong, pick the right one from the Category or Origin lists at the top — the tool reads the label again and refills the fields.
  3. Read through the filled-in fields and fix anything that doesn't match the approved application.
  4. Click Verify Against Label. Every field gets its own result, and the government warning is checked separately.
  5. Checking a whole stack of labels? Click Multiple labels under Upload mode on the left side of the page.

Files you can upload

  • PDF files and pictures (PNG, JPEG, or WebP).
  • Each file can be up to 10 MB — ordinary label photos and scans are almost always well under this.
  • One file can show every side of the label — for example, a PDF with the front and back on separate pages.
  • Drag a file into the upload box, or click the box to pick a file from your computer.

Checking many labels at once

  • Click Multiple labels under Upload mode, add as many files as you like, then click Process all. The tool checks them all at the same time.
  • Each file is checked as its own label.
  • If the front and back of one label are two separate pictures, give them matching names ending in _front and _back — for example airlie_front.jpg and airlie_back.jpg. The tool then checks them together as one product. (In Single label mode, put the back picture in the Second file box instead.)
  • Category and Origin are worked out automatically for each label, so you can check wine, malt beverages, and distilled spirits — imported or not — together. If a guess is wrong, fix it with the Category and Origin lists in that label's row.
  • Each row ends with one of two results: Pass or Needs attention. Pass means everything required is on the label, the government warning is word-for-word correct, and the other checks passed.
  • Click Details on a row to see the result for every field. Inside that panel, click Edit to open that label's fields on the main page, fix anything that's wrong, and click Verify Against Label again.
  • You can't upload a spreadsheet (CSV or Excel) yet — that may be added later.
