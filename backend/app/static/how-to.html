How To

App Swagger

Workflow

  1. Select the product category and origin.
  2. Upload the front label and optional back label.
  3. Click Extract Fields to populate the Expected COLA and Reviewed Label fields.
  4. Correct the Expected COLA fields to match the approved application.
  5. Click Verify Reviewed Fields and review the results.

Uploads

  • Accepted files: PDF, PNG, JPEG, and WebP.
  • Maximum file size: 10 MB per file.
  • Drag & Drop mode includes Select file for keyboard, mobile, and fallback use.

Notes

  • Government warning text is checked for exact wording, punctuation, and capitalization.
  • Bold type cannot be verified because extraction returns text, not typography metadata.
  • Batch upload support is planned next.
